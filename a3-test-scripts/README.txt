**How to run the automarker:**

1. Extract the scripts in this .zip file to a folder of your choosing.

2. Run mongodb as normal.

3. Run Neo4j as normal.

4. Run your profile-microservice and song-microservice.

5. Run AllTest_v2.*.py. The output as a result of the script is the automarker output.
